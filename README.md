# 1D Gaussian Wave Packet Simulator

An independent computational physics project built in Python to simulate and visualize the time-evolution and dispersion of a Gaussian wave packet. 

This project explores the intersection of numerical analysis, partial differential equations (PDEs), and scientific computing.

## Features & Implementation
- **Mathematical Modeling:** Solves the governing wave equations using numerical discretization over a spatial grid.
- **Efficient Computing:** Leverages vectorization techniques to handle matrix calculations efficiently.
- **Real-Time Visualization:** Generates graphical plots to track wave propagation, amplitude changes, and boundary interactions.

## Tech Stack
- **Language:** Python
- **Libraries:** NumPy, SciPy, Matplotlib
